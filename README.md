# MedQT: Ophthalmic Patient Record Management System

**MedQT** is a robust, specialized desktop application designed for managing medical patient records, with a primary focus on ophthalmic clinical practice. Built with Python and PySide6, it provides a modern and intuitive interface for healthcare professionals to track patient demographics, clinical examinations, and long-term treatment histories.

## 🚀 Key Features

*   **Comprehensive Patient Management:** Easily add, edit, and organize patient profiles and their complete medical history.
*   **Specialized Ophthalmic Records:** Tailored forms for primary examinations, including diagnostic data specifically structured for eye health.
*   **Clinical History Tracking:** Detailed management of "Stationary Cards" (inpatient records) and outpatient appointment history.
*   **Rich Diagnostic Database:** Integrated support for complex ophthalmic and comorbid diagnoses using flexible, template-based JSON data.
*   **Print and Report Generation:** Professional print preview and automated document generation for stationary cards and clinical summaries.
*   **Smart Search & Filtering:** Real-time search engine to quickly locate patients and specific medical records within the database.
*   **Wizard-Driven Workflows:** A "Create History Wizard" to guide practitioners through the clinical documentation process efficiently.
*   **Persistent SQLite Backend:** Reliable local data storage ensuring data integrity and fast access to patient information.

## 🛠️ Tech Stack

*   **Language:** Python 3.8+
*   **GUI Framework:** PySide6 (Qt for Python 6)
*   **Database:** SQLite3
*   **Data Handling:** JSON (for diagnostic templates and configurations)
*   **Reporting:** QtPrintSupport for document generation
